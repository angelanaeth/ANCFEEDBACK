Manual OAuth Authorization

Why Manual? The TrainingPeaks OAuth redirect URI is pre-registered to http://127.0.0.1:5000/... which doesn't match our sandbox URL. We'll copy the authorization code manually.
Steps to Authorize:
Step 1: Copy This OAuth URL
Step 2: Open in New Tab
Step 3: Authorize & Copy the Code

After logging in and authorizing:

  • You'll be redirected to: http://127.0.0.1:5000/handle_trainingpeaks_authorization?code=XXXXX
  • The page will fail to load (that's expected)
  • Copy the entire URL from your browser address bar

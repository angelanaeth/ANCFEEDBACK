## 🔄 How It Works

### For COACH Account (Current):

```
1. Coach connects TrainingPeaks account (OAuth)
   ↓
2. System fetches ALL athletes from TrainingPeaks API
   (93 athletes in your case)
   ↓
3. Coach selects any athlete from dropdown
   ↓
4. System fetches FRESH data for that athlete:
   - Historical workouts (90 days)
   - Future planned workouts (28 days)
   - Current week workouts (Mon-Sun)
   - Metrics, wellness, fueling data
   ↓
5. Dashboard displays athlete's data
   ↓
6. Click "Refresh" or select different athlete
   → Fetches fresh data again (no caching)
```

### For ATHLETE Account (Future/Optional):

```
1. Individual athlete connects their TrainingPeaks account
   ↓
2. System fetches THEIR data only
   ↓
3. Dashboard shows their own workouts/metrics
   ↓
4. Same refresh behavior (always fresh data)
```

5. Wait for "✅ Fueling Complete: 12 workouts"

### **Step 2: View Fueling Guidance**
1. Scroll down to **"💧 Fueling Guidance (Next Week)"** section
2. It will auto-expand after fueling is calculated
3. You'll see a table with:
   - Date
   - Workout name
   - Sport type
   - CHO, Fluid, Sodium values
   - **Copy button** for each workout

### **Step 3: Add to TrainingPeaks (Manual)**
For each workout:

1. Click **"Copy"** button next to the workout
2. Open TrainingPeaks
3. Find that workout in the athlete's calendar
4. Open the workout
5. Go to **"Pre-Activity Comments"** tab
6. **Paste** the fueling guidance
7. Save

### **Bulk Options**
- **Export to CSV**: Download all fueling as a spreadsheet

## 📊 Technical Details

### **What's Stored in Database:**
1. **users table**: Athlete profiles (weight_kg, cp_watts, cs_run_seconds, swim_pace_per_100, ftp, lactate_threshold_hr)
2. **tp_write_queue**: Fueling calculations per workout (workout_id, fuel_carb, fuel_fluid, fuel_sodium, status)

### **Fueling Calculations:**
- **Swim**: Base 40g CHO/hr
- **Bike**: Based on CP watts (higher CP = more CHO)
- **Run**: Based on CS pace (faster = more CHO)
- **Other**: Base 79g CHO/hr
- **Fluid**: 497ml/hr (constant)
- **Sodium**: 900mg/hr (constant)

### **Current Week Detection:**
- Calculates Monday-Sunday of current week
- Fetches all planned workouts in that range
- Updates/creates queue entries
- Status tracked: pending → (attempted) → failed (due to API limitations)

## ⚠️ TrainingPeaks API Limitations

### **What We Tried:**
❌ PUT /v2/workouts/id/{id} → 405 Method Not Allowed  
❌ PATCH /v2/workouts/id/{id} → 405 Method Not Allowed  
❌ POST /v1/workouts/{id} → 404 Not Found  
❌ POST /v1/workouts → 404 Not Found  
❌ POST /v1/workouts/plan → 404 Not Found  

### **Conclusion:**
The TrainingPeaks API **does NOT** allow:
- Creating workouts for athletes (coaches can only create for themselves)
- Updating planned workout fields (Title, Description, PreActivityComments, etc.)
- Adding notes to athlete calendars programmatically

### **What Coaches CAN Do:**
✅ Read athlete data  
✅ View athlete workouts  
✅ Attach/detach athletes  
✅ Search athletes
